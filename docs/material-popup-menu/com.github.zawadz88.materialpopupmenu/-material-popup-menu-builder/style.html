style

var style: Int

Style of the popup menu. Default is R.style.Widget_MPM_Menu.

For dark themes you should use R.style.Widget_MPM_Menu_Dark.

You can also provide your own style, however make sure that all of the attributes that are declared in R.style.Widget_MPM_Menu are also declared in your style.
